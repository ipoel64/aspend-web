👋 Pendahuluan

Selamat datang di Asisten Pendamping (Aspend). Aplikasi ini dikembangkan oleh hihang hoheng sebagai alat bantu bagi para Pendamping Program Keluarga Harapan (PKH) dalam melaksanakan tugas pendampingan sosial di lapangan.

Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, menyimpan, dan melindungi informasi Anda saat menggunakan aplikasi Aspend. Kami berkomitmen untuk melindungi privasi dan keamanan data Anda sesuai dengan peraturan yang berlaku.

Dengan mengunduh, menginstal, dan menggunakan aplikasi ini, Anda dianggap telah membaca, memahami, dan menyetujui seluruh ketentuan dalam Kebijakan Privasi ini.

📋 1. Data yang Dikumpulkan

Aplikasi Aspend mengumpulkan dan memproses beberapa jenis data untuk menjalankan fungsinya sebagai alat bantu pendampingan. Berikut adalah rincian data yang dikumpulkan:

Jenis Data Detail Penyimpanan
Akun Google Nama, alamat email, dan foto profil yang diperoleh melalui proses masuk (login) menggunakan Google Sign-In Google
Lokasi GPS Koordinat lokasi perangkat untuk keperluan geotagging (penandaan lokasi) pada laporan kunjungan Google Sheets
Foto & Gambar Foto yang diambil melalui kamera atau dipilih dari galeri sebagai bukti dokumentasi dalam laporan Google Drive
Rekaman Suara Input suara yang digunakan untuk fitur speech-to-text (konversi suara ke teks); audio tidak disimpan secara permanen Lokal (sementara)
Dokumen PDF Laporan dalam format PDF yang dihasilkan oleh aplikasi Google Drive Lokal
Data Laporan Data kunjungan, catatan pendampingan, dan informasi laporan yang diinput oleh pengguna Google Sheets
File CSV File data yang diimpor oleh pengguna dari Google Drive untuk keperluan pengolahan data Google Drive
API Key AI Kunci API layanan AI (Groq/OpenRouter) yang dimasukkan secara mandiri oleh pengguna untuk mengakses fitur kecerdasan buatan Lokal (perangkat)

🎯 2. Penggunaan Data

Data yang dikumpulkan digunakan secara eksklusif untuk tujuan berikut:

  • Autentikasi Pengguna: Data akun Google digunakan untuk proses masuk (login) dan identifikasi pengguna dalam aplikasi.
  • Pembuatan Laporan: Data lokasi GPS, foto, dan input teks/suara digunakan untuk menyusun laporan kunjungan pendampingan PKH.
  • Geotagging: Koordinat lokasi GPS ditambahkan ke laporan sebagai bukti kehadiran di lokasi kunjungan.
  • Dokumentasi Visual: Foto dari kamera atau galeri digunakan sebagai bukti pendukung dalam laporan.
  • Input Teks Melalui Suara: Mikrofon digunakan untuk fitur speech-to-text guna mempermudah penginputan data di lapangan.
  • Penyimpanan & Pengelolaan Dokumen: Google Drive digunakan untuk menyimpan laporan PDF dan mengimpor file CSV.
  • Pencatatan Data: Google Sheets digunakan untuk menyimpan dan mengelola data laporan pendampingan.
  • Fitur AI: API key yang disediakan pengguna digunakan untuk mengakses layanan AI pihak ketiga (Groq/OpenRouter) guna mendukung fitur kecerdasan buatan dalam aplikasi.

✅ Kami TIDAK menggunakan data Anda untuk keperluan iklan, pemasaran, pembuatan profil, atau tujuan komersial lainnya.

💾 3. Penyimpanan Data

Aspend dirancang dengan prinsip privasi berdasarkan desain (privacy by design). Aplikasi ini tidak memiliki server eksternal milik pengembang. Seluruh data disimpan di:

📱 Penyimpanan Lokal (Perangkat Pengguna)

  • API key AI (Groq/OpenRouter) disimpan secara terenkripsi di penyimpanan lokal perangkat.
  • File PDF laporan dapat disimpan di penyimpanan internal perangkat.
  • Data sementara (cache) dari proses speech-to-text disimpan sementara dan dihapus setelah proses selesai.

☁️ Penyimpanan Cloud (Akun Google Pengguna)

  • Dokumen PDF laporan disimpan di Google Drive milik pengguna sendiri.
  • Data laporan dan catatan pendampingan disimpan di Google Sheets milik pengguna sendiri.
  • Foto dokumentasi diunggah ke Google Drive milik pengguna sendiri.

✅ Pengembang TIDAK memiliki akses ke data yang tersimpan di akun Google Drive atau Google Sheets milik pengguna. Semua data sepenuhnya berada di bawah kendali pengguna.

🤝 4. Berbagi Data dengan Pihak Ketiga

Aspend tidak menjual, memperdagangkan, atau membagikan data pribadi pengguna kepada pihak ketiga mana pun, kecuali interaksi yang diperlukan dengan layanan berikut:

Layanan Tujuan Penggunaan Data yang Dibagikan
Google Sign-In Autentikasi dan proses masuk pengguna Nama, email, foto profil
Google Drive API Penyimpanan dokumen PDF dan impor file CSV File dokumen pengguna
Google Sheets API Penyimpanan dan pengelolaan data laporan Data laporan pendampingan
Groq / OpenRouter Pemrosesan fitur kecerdasan buatan (AI) Teks input pengguna (menggunakan API key milik pengguna sendiri)

⚠️ Penggunaan layanan Google tunduk pada Kebijakan Privasi Google. Penggunaan layanan Groq/OpenRouter tunduk pada kebijakan privasi masing-masing penyedia layanan tersebut.

🔒 5. Keamanan Data

Kami menerapkan langkah-langkah keamanan berikut untuk melindungi data Anda:

  • Autentikasi Aman: Proses login menggunakan protokol OAuth 2.0 melalui Google Sign-In, sehingga aplikasi tidak pernah menyimpan kata sandi pengguna.
  • Penyimpanan Lokal Terenkripsi: API key AI disimpan secara terenkripsi di penyimpanan lokal perangkat dan tidak dapat diakses oleh aplikasi lain.
  • Tanpa Server Eksternal: Aplikasi tidak mengirimkan data ke server milik pengembang. Semua data tersimpan di perangkat pengguna atau akun Google milik pengguna sendiri.
  • Koneksi Terenkripsi: Seluruh komunikasi dengan layanan Google dan penyedia AI menggunakan protokol HTTPS yang terenkripsi.
  • Izin Minimal: Aplikasi hanya meminta izin (permission) yang benar-benar diperlukan untuk menjalankan fungsinya.

⚠️ Meskipun kami berupaya maksimal untuk melindungi data Anda, tidak ada metode transmisi melalui internet atau penyimpanan elektronik yang 100% aman. Kami menyarankan pengguna untuk menjaga keamanan akun Google dan perangkat masing-masing.

⚙️ 6. Izin Aplikasi (Permissions)

Aplikasi Aspend memerlukan beberapa izin perangkat untuk dapat berfungsi dengan baik. Berikut penjelasan untuk setiap izin yang diminta:

Izin Alasan Penggunaan
Kamera Mengambil foto sebagai bukti dokumentasi dalam laporan kunjungan pendampingan.
Lokasi (GPS) Memperoleh koordinat lokasi untuk geotagging pada laporan sebagai bukti kehadiran di lokasi.
Mikrofon Merekam suara untuk fitur speech-to-text (konversi suara ke teks) guna mempermudah penginputan data.
Penyimpanan / Media Menyimpan file PDF laporan ke penyimpanan internal perangkat dan mengakses foto dari galeri.
Internet Berkomunikasi dengan layanan Google (Drive, Sheets, Sign-In) dan layanan AI (Groq/OpenRouter).

Anda dapat mencabut izin-izin di atas kapan saja melalui menu Pengaturan > Aplikasi > Aspend > Izin pada perangkat Anda. Namun, pencabutan izin tertentu dapat menyebabkan beberapa fitur aplikasi tidak berfungsi sebagaimana mestinya.

✊ 7. Hak-Hak Pengguna

Sebagai pengguna aplikasi Aspend, Anda memiliki hak-hak berikut:

  • Hak Akses: Anda berhak mengetahui data apa saja yang dikumpulkan dan bagaimana data tersebut digunakan.
  • Hak Penghapusan: Anda dapat menghapus seluruh data Anda kapan saja. Karena data disimpan di akun Google dan perangkat Anda sendiri, Anda memiliki kendali penuh untuk menghapusnya:
    • Hapus file di Google Drive Anda
    • Hapus spreadsheet di Google Sheets Anda
    • Hapus data lokal melalui pengaturan aplikasi atau dengan menghapus instalasi (uninstall) aplikasi
  • Hak Penarikan Persetujuan: Anda dapat mencabut izin aplikasi atau menghapus akses aplikasi ke akun Google Anda kapan saja melalui pengaturan keamanan akun Google.
  • Hak Portabilitas: Data Anda yang tersimpan di Google Drive dan Google Sheets sudah dalam format standar (PDF, CSV, Spreadsheet) yang dapat diunduh dan dipindahkan kapan saja.
  • Hak Pembatasan: Anda berhak membatasi pengolahan data dengan cara mencabut izin tertentu pada perangkat Anda.

👶 8. Perlindungan Anak

Aplikasi Aspend tidak ditujukan untuk anak-anak di bawah usia 13 tahun. Kami tidak secara sengaja mengumpulkan informasi pribadi dari anak-anak di bawah 13 tahun.

Aplikasi ini dirancang khusus untuk digunakan oleh tenaga Pendamping PKH yang merupakan orang dewasa dalam kapasitas profesional mereka. Jika kami mengetahui bahwa kami telah mengumpulkan data pribadi dari anak di bawah 13 tahun, kami akan segera mengambil langkah untuk menghapus informasi tersebut.

📝 9. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Setiap perubahan akan dipublikasikan pada halaman ini dengan tanggal efektif yang diperbarui.

Kami menyarankan pengguna untuk meninjau halaman ini secara berkala agar tetap mengetahui informasi terbaru mengenai praktik privasi kami. Perubahan pada Kebijakan Privasi ini berlaku efektif segera setelah dipublikasikan pada halaman ini.

Jika terdapat perubahan signifikan pada cara kami memperlakukan data pribadi pengguna, kami akan memberikan pemberitahuan melalui pembaruan aplikasi atau notifikasi di dalam aplikasi.
